This is just a script that organise bookmarks export to html in some browser, the result is a page that can be parse for searching some word.

## How to Use

### Exporting Bookmarks

#### Firefox
1. Click the menu button (☰) in the top-right corner
2. Go to **Bookmarks** → **Manage Bookmarks** (or press Ctrl+Shift+B)
3. In the Bookmarks Manager, click the menu icon and select **Import and Backup** → **Export Bookmarks to HTML**
4. Choose a location to save the file and name it (e.g., `bookmarks.html`)

#### Chrome
1. Click the menu button (⋮) in the top-right corner
2. Go to **Bookmarks and lists** → **Bookmark manager**
3. Click the menu icon (⋮) and select **Export bookmarks**
4. Choose a location to save the file

#### Brave
1. Click the menu button (☰) in the top-right corner
2. Go to **Bookmarks** → **Bookmark manager**
3. Click the menu icon (⋮) and select **Export bookmarks**
4. Choose a location to save the file

### Using the Organized Bookmarks Page

1. Open the exported `bookmarks.html` file in your browser
2. Use the search functionality to find specific bookmarks by keyword
3. Browse through organized categories for easy navigation
4. Bookmarks are displayed in a clean, organized layout for quick access

### Features

- ✅ Works with Firefox, Chrome, and Brave
- ✅ Searchable bookmark page
- ✅ Organized by categories
- ✅ Clean and user-friendly interface
- ✅ No external dependencies required
